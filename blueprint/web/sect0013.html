13 Def 6: Gauged Stabilizers (Vertex and Plaquette Terms)

This chapter defines the stabilizer generators for the gauged code: vertex stabilizers (Gauss law generators) \(\mathbb {A}_v\) and plaquette stabilizers (flux terms) \(\mathbb {B}_p\). It also establishes the relationship between plaquette boundaries and cycles in the chain complex.

Definition 13.1 Gauged Cell Complex
✓
#

Uses

  • Definition 2.1

Lean declarations

  • GaugingLDPC.GaugedCellComplex

A gauged cell complex \(\mathcal{G} = (\mathcal{G}_0, \mathcal{G}_1, \mathcal{G}_2)\) over types \(V\) (vertices), \(E\) (edges), and \(P\) (plaquettes), with \(E\) finite, extends an oriented graph structure with the following additional data:

  • Plaquette boundary function: A function \(\texttt{plaquetteBoundary} : P \to E \to \mathbb {Z}_w\) giving the coefficient of edge \(e\) in the boundary of plaquette \(p\). This coefficient is typically \(0\), \(1\), or \(-1 \pmod{w}\).

  • Boundary cycle condition: For all plaquettes \(p \in P\) and vertices \(v \in V\):

    \[ \sum _{e \in E} \texttt{plaquetteBoundary}(p, e) \cdot \delta _v(e) = 0 \]

    where \(\delta _v(e) = 1\) if \(v = \texttt{target}(e)\), \(\delta _v(e) = -1\) if \(v = \texttt{source}(e)\), and \(\delta _v(e) = 0\) otherwise.

This closure condition ensures that plaquette boundaries are actual cycles, i.e., \(\partial _1(\partial _2(p)) = 0\).

Definition 13.7 Gauged System State
✓
#

Lean declarations

  • GaugingLDPC.GaugedSystemState

The state of the full gauged system with vertices \(V\), edges \(E\), and matter qudits \(M\) is defined as:

\[ \texttt{GaugedSystemState}(w, V, E, M) := (V \to \mathbb {Z}_w) \to (E \to \mathbb {Z}_w) \to (M \to \mathbb {Z}_w) \to \mathbb {C} \]

A state assigns complex amplitudes to configurations \((f_V, f_E, f_M)\) where \(f_V\), \(f_E\), and \(f_M\) are functions assigning \(\mathbb {Z}_w\) values to vertices, edges, and matter qudits respectively.

Definition 13.8 Vertex Stabilizer
✓
#

Uses

  • Definition 13.7
  • Definition 13.2
  • Definition 12.19

Lean declarations

  • GaugingLDPC.vertexStabilizer

The vertex stabilizer (Gauss law generator) \(\mathbb {A}_v\) for vertex \(v\) is defined as:

\[ \mathbb {A}_v := \mathsf{T}_{\Sigma (v)} \prod _{e \in \mathcal{G}_1} (\mathcal{X}_e)^{(\partial _1^{\mathcal{G}})_{v,e}} \]

where \(\mathsf{T}_{\Sigma (v)}\) is the on-site symmetry restricted to the vertex set \(\Sigma (v)\), and \(\mathcal{X}_e\) is the generalized Pauli-\(X\) on edge qudit \(e\).

The action on a state \(\Psi \) is given by:

\[ (\mathbb {A}_v \cdot \Psi )(f_V, f_E, f_M) = \Psi (f_V, f_E', f_M') \]

where:

\begin{align*} f_E’(e) & = f_E(e) - (\partial _1)_{v,e} \\ f_M’(m) & = \begin{cases} f_M(m) - 1 & \text{if } m \in \Sigma (v) \\ f_M(m) & \text{otherwise} \end{cases}\end{align*}

Definition 13.10 Plaquette Z-Phase
✓
#

Uses

  • Definition 13.3
  • Definition 12.4

Lean declarations

  • GaugingLDPC.plaquetteZPhase

The plaquette Z-phase for plaquette \(p\) given edge configuration \(f_E\) is:

\[ \texttt{plaquetteZPhase}(p, f_E) := \omega ^{\left(\sum _{e \in E} (\partial _2)_{p,e} \cdot f_E(e)\right) \bmod w} \]

where \(\omega = e^{2\pi i/w}\) is the primitive \(w\)-th root of unity.

Definition 13.11 Plaquette Stabilizer
✓
#

Uses

  • Definition 13.10

Lean declarations

  • GaugingLDPC.plaquetteStabilizer

The plaquette stabilizer (flux term) \(\mathbb {B}_p\) for plaquette \(p\) is defined as:

\[ \mathbb {B}_p := \prod _{e \in p} \mathcal{Z}_e^{(\partial _2^{\mathcal{G}})_{p,e}} \]

The action on a state \(\Psi \) is given by:

\[ (\mathbb {B}_p \cdot \Psi )(f_V, f_E, f_M) = \omega ^{\sum _e (\partial _2)_{p,e} \cdot f_E(e)} \cdot \Psi (f_V, f_E, f_M) \]

where \(\omega = e^{2\pi i/w}\).

The boundary of plaquette \(p\) as a 1-chain in \(E \to _0 \mathbb {Z}_w\) is:

Theorem 13.15 Plaquette Boundary is a Cycle
✓
#

Uses

  • GaugingLDPC.plaquette_boundary_is_cycle

The boundary of a plaquette is a 1-cycle: \(\partial _1(\partial _2(p)) = 0\). Specifically, for all vertices \(v\):

\[ \sum _{e \in E} (\partial _2(p))_e \cdot (\partial _1)_{v,e} = 0 \]
Proof ▶

Let \(v\) be an arbitrary vertex. By simplification and unfolding definitions, this reduces to the boundary_cycle axiom of the gauged cell complex structure, which states exactly that plaquette boundaries satisfy the cycle condition.

Theorem 13.16 Plaquette Corresponds to Cycle
✓
#

Uses

  • Definition 13.14
  • Definition 11.1
  • Definition 13.14
  • Definition 13.3
  • Definition 11.1
  • Definition 13.1

Lean declarations

  • GaugingLDPC.plaquette_corresponds_to_cycle

The plaquette boundary lies in the kernel of \(\partial _1\) (the 1-cycle space \(Z_1\)):

\[ \partial _1(\partial _2(p)) = 0 \]

This confirms that \(\mathbb {B}_p\) operators correspond to cycles in \(Z_1(\mathcal{G}, \mathbb {F}_w)\).

Proof ▶

By extensionality, it suffices to show equality at each vertex \(v\). We unfold the definitions of boundaryMapZMod and plaquetteBoundaryChain. The linear combination evaluates to a sum over edges.

For each edge \(e\), the boundary map contribution at \(v\) is:

\[ (\delta _{\texttt{target}(e)} - \delta _{\texttt{source}(e)})(v) = \begin{cases} 1 & \text{if } v = \texttt{target}(e) \\ -1 & \text{if } v = \texttt{source}(e) \\ 0 & \text{otherwise} \end{cases} \]

We establish this formula by case analysis on whether \(v\) equals the target or source of \(e\). The sum over edges then equals \(\sum _e (\partial _2)_{p,e} \cdot \delta _v(e)\), which is zero by the boundary_cycle axiom of the gauged cell complex. We use subset summation to extend from the support to all of \(E\), noting that terms outside the support contribute zero.

Theorem 13.19 Boundary Space is Contained in Cycle Space
✓
#

Uses

  • Definition 13.18
  • Definition 13.17
  • Theorem 13.16
  • Definition 13.18
  • Theorem 13.16
  • Definition 13.17

Lean declarations

  • GaugingLDPC.boundarySpace_le_cycleSpace

All boundaries are cycles: \(B_1 \subseteq Z_1\). This follows from the chain complex property \(\partial _1 \circ \partial _2 = 0\).

Proof ▶

We rewrite \(B_1\) as a span and apply the span containment lemma. Let \(c\) be an element of the range of plaquetteBoundaryChain, so \(c = \partial _2(p)\) for some plaquette \(p\). We obtain this \(p\) from the set membership. By the membership characterization of the kernel and Theorem 13.16, we have \(\partial _1(c) = \partial _1(\partial _2(p)) = 0\), so \(c \in \ker (\partial _1) = Z_1\).
