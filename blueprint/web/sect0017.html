17 Def 9: Gauging Graph Criteria

This chapter formalizes the four criteria that a valid gauging graph \(\mathcal{G}\) must satisfy for the gauging procedure. These criteria ensure bounded operator weights and proper code distance preservation.

Definition 17.1 Gauged Cell Complex to Simple Graph
✓
#

Uses

  • Definition 13.1

Lean declarations

  • GaugingLDPC.GaugedCellComplex.toSimpleGraph

Given a gauged cell complex \(G\), we define the underlying simple graph by forgetting orientation: two vertices \(v_1, v_2\) are adjacent if \(v_1 \neq v_2\) and there exists an edge \(e\) such that either \((s(e) = v_1 \land t(e) = v_2)\) or \((s(e) = v_2 \land t(e) = v_1)\), where \(s\) and \(t\) denote the source and target functions respectively.

Definition 17.2 Maximum Degree
✓
#

Lean declarations

  • GaugingLDPC.maxDegree

For a finite simple graph \(G\) on a nonempty vertex set \(V\), the maximum degree is defined as:

\[ \mathrm{maxDegree}(G) := \sup _{v \in V} \deg (v) \]
Definition 17.3 Bounded Degree (Criterion 1)
✓
#

Uses

  • Definition 17.2

Lean declarations

  • GaugingLDPC.BoundedDegree

A graph \(G\) has bounded degree with bound \(d \in \mathbb {N}\) if:

\[ \mathrm{maxDegree}(G) \leq d \]

This ensures the weight of vertex stabilizer terms \(\mathbb {A}_v\) is bounded: \(|\mathrm{Supp}(\mathbb {A}_v)| = O(1)\).

Theorem 17.4 Degree Bounded by Maximum
✓
#

Uses

  • Definition 17.2
  • Definition 17.2

Lean declarations

  • GaugingLDPC.degree_le_maxDegree

For any vertex \(v\) in a finite nonempty graph \(G\):

\[ \deg (v) \leq \mathrm{maxDegree}(G) \]
Proof ▶

By definition of maximum degree as the supremum over all vertices, and since \(v \in V\), we have \(\deg (v) \leq \sup _{u \in V} \deg (u) = \mathrm{maxDegree}(G)\). This follows directly from the property of supremum over a finite set containing \(v\).

Theorem 17.5 Bounded Degree at Every Vertex
✓
#

Uses

  • Definition 17.3
  • Theorem 17.4
  • Theorem 17.4
  • Definition 17.3

Lean declarations

  • GaugingLDPC.BoundedDegree.degree_at_vertex

If \(G\) has bounded degree with bound \(d\), then for every vertex \(v\):

\[ \deg (v) \leq d \]
Proof ▶

By transitivity of \(\leq \): we have \(\deg (v) \leq \mathrm{maxDegree}(G)\) from Theorem 17.4, and \(\mathrm{maxDegree}(G) \leq d\) from the bounded degree hypothesis. Therefore \(\deg (v) \leq d\).

Definition 17.6 Incident Edges
✓
#

Uses

  • Definition 13.1

Lean declarations

  • GaugingLDPC.incidentEdges

The incident edges of a vertex \(v\) in a gauged cell complex \(G\) are:

\[ \mathrm{incidentEdges}(G, v) := \{ e \in E \mid s(e) = v \lor t(e) = v\} \]

where \(s\) and \(t\) are the source and target functions.

Definition 17.7 Vertex Stabilizer Support Bound
✓
#

Uses

  • Definition 17.6
  • Definition 12.19

Lean declarations

  • GaugingLDPC.vertexStabilizerSupportBound

The support weight bound for the vertex stabilizer \(\mathbb {A}_v\) is:

\[ |\Sigma (v)| + |\mathrm{incidentEdges}(G, v)| \]

where \(\Sigma (v)\) is the matter assignment at vertex \(v\). This accounts for the on-site symmetry acting on \(\Sigma (v)\) and the \(X\) operators on all edges incident to \(v\).

Definition 17.8 Plaquette Boundary Weight
✓
#

Uses

  • Definition 13.1

Lean declarations

  • GaugingLDPC.plaquetteBoundaryWeight

The weight of a plaquette boundary is the number of edges with non-zero coefficient:

\[ \mathrm{plaquetteBoundaryWeight}(G, p) := |\{ e \in E \mid \partial _2(p)(e) \neq 0\} | \]
Definition 17.9 Sparse Cycle Basis (Criterion 2)
✓
#

Uses

  • Definition 17.8
  • Definition 13.20

Lean declarations

  • GaugingLDPC.SparseCycleBasis

A gauged cell complex \(G\) has a sparse cycle basis with bound \(d\) if:

  1. Every plaquette boundary has weight at most \(d\): \(\forall p \in P\), \(\mathrm{plaquetteBoundaryWeight}(G, p) \leq d\)

  2. The plaquette boundaries generate all cycles, i.e., \(H^1(\mathcal{G}) = 0\)

This ensures \(|\mathrm{Supp}(\mathbb {B}_p)| = O(1)\) for each plaquette \(p\).

Definition 17.10 Plaquette Stabilizer Support
✓
#

Uses

  • Definition 13.1

Lean declarations

  • GaugingLDPC.plaquetteStabilizerSupport

The support of the plaquette stabilizer \(\mathbb {B}_p\) consists of all edges in its boundary:

\[ \mathrm{plaquetteStabilizerSupport}(G, p) := \{ e \in E \mid \partial _2(p)(e) \neq 0\} \]
Theorem 17.11 Plaquette Stabilizer Support Card
✓
#

Uses

  • Definition 17.10
  • Definition 17.8
  • Definition 17.10
  • Definition 17.8

Lean declarations

  • GaugingLDPC.plaquetteStabilizerSupport_card

The cardinality of the plaquette stabilizer support equals the plaquette boundary weight:

\[ |\mathrm{plaquetteStabilizerSupport}(G, p)| = \mathrm{plaquetteBoundaryWeight}(G, p) \]
Proof ▶

This holds by reflexivity, as both are defined as the cardinality of the same set \(\{ e \in E \mid \partial _2(p)(e) \neq 0\} \).

Theorem 17.12 Sparse Cycles Implies Bounded Plaquette Term
✓
#

Uses

If \(G\) has a sparse cycle basis with bound \(d\), then for every plaquette \(p\):

Proof ▶

By rewriting using Theorem 17.11, we have \(|\mathrm{plaquetteStabilizerSupport}(G, p)| = \mathrm{plaquetteBoundaryWeight}(G, p)\). The result then follows directly from the sparse cycle basis hypothesis, which states that \(\mathrm{plaquetteBoundaryWeight}(G, p) \leq d\) for all plaquettes \(p\).

Definition 17.15 Stabilizer Generator
✓
#

Uses

  • Definition 14.1
  • Definition 11.3
  • Definition 2.1

Lean declarations

  • GaugingLDPC.StabilizerGenerator

A stabilizer generator consists of:

  • A symmetrized stabilizer \(\bar{S}\)

  • The set of charge configurations \(C_0(S) := \{ \chi : \llbracket S \rrbracket _\chi \neq 0\} \)

  • A proof that all charge configurations in \(C_0(S)\) are exact (lie in \(\mathrm{im}(\partial _1)\))

Definition 17.16 Constant Charge Neutralization Path (Criterion 3)
✓
#

Uses

  • Definition 17.15
  • Definition 17.14

A gauged cell complex \(G\) with stabilizer generators satisfies the constant charge neutralization path property with bound \(d\) if:

\[ \forall S \in \mathrm{stabilizers}, \forall \chi \in C_0(S), \exists \alpha : \partial _1(\alpha ) = \chi \land \mathrm{chainWeight}(\alpha ) \leq d \]

This ensures every decorated stabilizer \(\mathbb {S}\) has bounded weight.

Definition 17.19 Valid Gauging Graph
✓
#

Uses

  • Definition 13.1
  • Definition 17.3

  • Definition 12.19
  • Definition 17.15

Lean declarations

  • GaugingLDPC.ValidGaugingGraph

A valid gauging graph is a gauged cell complex \(G\) together with:

  • A degree bound constant \(d_{\deg }\)

  • A cycle weight bound constant \(d_{\mathrm{cycle}}\)

  • A path weight bound constant \(d_{\mathrm{path}}\)

satisfying all four criteria:

  1. Criterion 1 (Bounded Degree): \(\mathrm{maxDegree}(G) \leq d_{\deg }\)

  2. Criterion 2 (Sparse Cycles): \(G\) has a sparse cycle basis with bound \(d_{\mathrm{cycle}}\)

  3. Criterion 3 (Constant Path): Neutralization paths have weight at most \(d_{\mathrm{path}}\)

  4. Criterion 4 (Expansion): \(h(G) \geq 1\)

Theorem 17.23 Small Subset Has Large Boundary
✓

  • GaugingLDPC.ValidGaugingGraph.small_subset_has_large_boundary

If \(G\) is a valid gauging graph and \(U\) is a small subset of vertices, then:

\[ |U| \leq |\partial U| \]

Theorem 17.24 Cheeger Constant Implies Distance Bound

For a valid gauging graph \(G\) and any small subset \(U\) of vertices:
