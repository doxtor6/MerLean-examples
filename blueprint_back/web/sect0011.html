11 Def 4: Gauging (Decoration Map)

This chapter develops the gauging (decoration) map, which attaches gauge field operators to matter operators based on their charge content. Given an exact charge distribution \(\boldsymbol {\chi } \in \partial C_1\) (meaning \(\boldsymbol {\chi } = \partial _1 \boldsymbol {\alpha }\) for some 1-chain \(\boldsymbol {\alpha }\)) and a spanning tree \(T_1 \subset C_1\) that generates all of \(\partial C_1\), the gauging map decorates charge components with appropriate gauge \(Z\)-operators.

11.1 Chain Complex over ZMod p

We work with an oriented graph \(G\) with vertices \(V\) (sites, \(C_0\)) and edges \(E\) (gauge qudits, \(C_1\)). The boundary map \(\partial _1: C_1 \to C_0\) sends each edge to its target minus its source.

Definition 11.1 Boundary Map over \(\mathbb {Z}/p\mathbb {Z}\)
✓
#

Lean declarations

  • GaugingLDPC.boundaryMapZMod

The boundary map \(\partial _1: C_1 \to C_0\) over \(\mathbb {Z}/p\mathbb {Z}\) is the linear map

\[ \partial _1 : (E \to _0 \mathbb {Z}/p\mathbb {Z}) \to (V \to _0 \mathbb {Z}/p\mathbb {Z}) \]

defined by sending each edge \(e\) to \(\mathbf{1}_{\mathrm{target}(e)} - \mathbf{1}_{\mathrm{source}(e)}\), where \(\mathbf{1}_v\) denotes the indicator function supported at vertex \(v\).

11.2 Exact Charge Distributions

Definition 11.2 Exact Charge Submodule
✓
#

Uses

  • Definition 11.1

Lean declarations

  • GaugingLDPC.ExactChargeSubmodule

The exact charge submodule \(\partial C_1\) is the image of the boundary map:

\[ \partial C_1 := \mathrm{im}(\partial _1) = \{ \chi \in C_0 : \exists \alpha \in C_1, \partial _1(\alpha ) = \chi \} . \]

Elements of this submodule are called exact charge distributions—these are the charges that can be “gauged away” by decoration.

A charge distribution \(\chi : V \to _0 \mathbb {Z}/p\mathbb {Z}\) is exact if \(\chi \in \mathrm{im}(\partial _1)\).

Theorem 11.4 Characterization of Exact Charges
✓
#

11.3 Spanning Basis

Definition 11.5 Spanning Basis
✓
#

Uses

  • Definition 11.2
  • Definition 11.1

Lean declarations

  • GaugingLDPC.SpanningBasis

A spanning basis for the gauging construction is a structure consisting of:

  • A submodule \(T_1 \subseteq C_1\) (the “spanning tree” subspace),

  • A restricted boundary map \(\partial _1|_{T_1} : T_1 \to \partial C_1\),

  • A proof that the restriction equals the original boundary on \(T_1\),

  • A proof that the restriction is surjective (\(T_1\) generates all exact charges),

  • A proof that the restriction is injective (unique preimages).

The key property is that \(\partial _1|_{T_1} : T_1 \to \partial C_1\) is bijective, ensuring unique lifting of exact charges.

Theorem 11.6 Bijectivity of Restricted Boundary
✓
#

Uses

  • Definition 11.5
  • Definition 11.5

Lean declarations

  • GaugingLDPC.SpanningBasis.bijective

For a spanning basis \(B\), the restricted boundary map \(\partial _1|_{T_1}\) is bijective.

Proof ▶

This holds by construction: bijectivity is the conjunction of injectivity and surjectivity, both of which are required fields of the spanning basis structure.

Definition 11.7 Alpha of Chi
✓
#

Uses

  • Definition 11.5

Lean declarations

  • GaugingLDPC.SpanningBasis.alphaOfChi

For a spanning basis \(B\) and an exact charge \(\chi \in \partial C_1\), the function \(\alpha (\chi ) : E \to _0 \mathbb {Z}/p\mathbb {Z}\) is defined as the unique element of \(T_1\) satisfying \(\partial _1(\alpha ) = \chi \). This is obtained by taking the inverse function of the restricted boundary map and extracting the underlying 1-chain.

Theorem 11.8 Alpha of Chi Membership
✓
#

Uses

  • Definition 11.7
  • Definition 11.5
  • Definition 11.7

For any exact charge \(\chi \), the 1-chain \(\alpha (\chi )\) lies in \(T_1\).

Proof ▶

By definition, \(\alpha (\chi )\) is the value of the inverse function applied to \(\chi \), and this inverse returns elements of \(T_1\) (as a subtype). The membership follows from the property field of the subtype.

Theorem 11.9 Alpha of Chi Specification
✓
#

Uses

  • Definition 11.7
  • Definition 11.1
  • Definition 11.7
  • Definition 11.5

Lean declarations

  • GaugingLDPC.SpanningBasis.alphaOfChi_spec

For any exact charge \(\chi \), we have \(\partial _1(\alpha (\chi )) = \chi \).

Proof ▶

Unfolding the definition of \(\alpha (\chi )\), we use that the inverse function is a right inverse of the restricted boundary (by surjectivity). Let \(h\) be the fact that the inverse function applied to \(\chi \) maps back to \(\chi \) under the restricted boundary. By the restriction property, the restricted boundary of an element equals the original boundary applied to its underlying value. The result follows by applying congruence under the subtype coercion.

Theorem 11.10 Uniqueness of Alpha of Chi
✓
#

Uses

  • Definition 11.7
  • Definition 11.5
  • Definition 11.1
  • Definition 11.7
  • Definition 11.5

Lean declarations

  • GaugingLDPC.SpanningBasis.alphaOfChi_unique

For any exact charge \(\chi \) and any \(\alpha ' \in T_1\) satisfying \(\partial _1(\alpha ') = \chi \), we have \(\alpha ' = \alpha (\chi )\).

Proof ▶

We first show that \(\partial _1|_{T_1}(\alpha ') = \chi \) by applying subtype extensionality and using the restriction property together with the hypothesis. Then, using that the inverse function is a left inverse (by injectivity), we have that the inverse of \(\partial _1|_{T_1}(\alpha ')\) equals \(\alpha '\). Rewriting with our established equality gives the result by congruence under subtype coercion.

Theorem 11.11 Alpha of Zero
✓
#

Uses

The zero exact charge lifts to the zero 1-chain: \(\alpha (0) = 0\).

Proof ▶

First, \(0 \in T_1\) since \(T_1\) is a submodule. Second, \(\partial _1(0) = 0\) since \(\partial _1\) is a linear map. By uniqueness (Theorem 11.10), any element of \(T_1\) mapping to \(0\) must equal \(\alpha (0)\). Applying this to the zero chain gives \(\alpha (0) = 0\).

11.4 Gauge Z-Operators

Definition 11.12 Gauge Z-Basis
✓
#

Lean declarations

  • GaugingLDPC.GaugeZBasis

A gauge Z-basis for prime \(p\) and edge set \(E\) consists of:

  • A function \(Z : E \to O_{\mathrm{gauge}}\) assigning a \(Z\)-operator to each edge,

  • A proof that \((Z_e)^p = 1\) for all edges \(e\) (the \(p\)-th root of unity property).

Here \(Z_e\) is the generalized Pauli-\(Z\) operator on the gauge qudit at edge \(e\).

Definition 11.13 Z Power
✓
#

Uses

  • Definition 11.12

Lean declarations

  • GaugingLDPC.GaugeZBasis.ZPow

For a gauge Z-basis, edge \(e\), and exponent \(k \in \mathbb {Z}/p\mathbb {Z}\), define

\[ Z_e^k := (Z_e)^{\mathrm{val}(k)} \]

where \(\mathrm{val}(k) \in \{ 0, 1, \ldots , p-1\} \) is the canonical representative of \(k\).

Theorem 11.14 Z Power Zero
✓
#

Uses

  • Definition 11.13
  • Definition 11.13

Lean declarations

  • GaugingLDPC.GaugeZBasis.ZPow_zero

For any edge \(e\), \(Z_e^0 = 1\).

Proof ▶

By simplification: \(Z_e^0 = (Z_e)^{\mathrm{val}(0)} = (Z_e)^0 = 1\).

Theorem 11.15 Z Power Periodicity
✓
#

Uses

  • Definition 11.12
  • Definition 11.12

Lean declarations

  • GaugingLDPC.GaugeZBasis.ZPow_periodic

For any edge \(e\) and natural number \(k\), \((Z_e)^k = (Z_e)^{k \mod p}\).

Proof ▶

Rewrite \(k = (k \div p) \cdot p + (k \mod p)\). Then \((Z_e)^k = (Z_e)^{(k \div p) \cdot p + (k \mod p)} = (Z_e)^{(k \div p) \cdot p} \cdot (Z_e)^{k \mod p} = ((Z_e)^p)^{k \div p} \cdot (Z_e)^{k \mod p} = 1^{k \div p} \cdot (Z_e)^{k \mod p} = (Z_e)^{k \mod p}\).

Theorem 11.16 Z Power Addition
✓
#

Uses

  • Definition 11.13
  • Theorem 11.15
  • Definition 11.13
  • Theorem 11.15

Lean declarations

  • GaugingLDPC.GaugeZBasis.ZPow_add

For any edge \(e\) and exponents \(k_1, k_2 \in \mathbb {Z}/p\mathbb {Z}\),

\[ Z_e^{k_1 + k_2} = Z_e^{k_1} \cdot Z_e^{k_2}. \]
Proof ▶

Expanding the definition of \(Z_e^k\) and using \((Z_e)^a \cdot (Z_e)^b = (Z_e)^{a+b}\), we need to show \((Z_e)^{\mathrm{val}(k_1+k_2)} = (Z_e)^{\mathrm{val}(k_1)+\mathrm{val}(k_2)}\). By periodicity, both sides equal \((Z_e)^{(\mathrm{val}(k_1)+\mathrm{val}(k_2)) \mod p}\), and by the definition of addition in \(\mathbb {Z}/p\mathbb {Z}\), this equals \((Z_e)^{\mathrm{val}(k_1+k_2)}\).

Definition 11.17 Gauge Z-Operator
✓
#

Uses

  • Definition 11.12
  • Definition 11.13

Lean declarations

  • GaugingLDPC.gaugeZOperator

For a 1-chain \(\alpha : E \to _0 \mathbb {Z}/p\mathbb {Z}\), the gauge Z-operator is

\[ \mathcal{Z}(\alpha ) := \prod _{e \in E} Z_e^{\alpha _e} \]

where \(\alpha _e\) denotes the coefficient of edge \(e\) in \(\alpha \).

Theorem 11.18 Gauge Z-Operator of Zero
✓
#

Uses

  • Definition 11.17
  • Theorem 11.14
  • Definition 11.17
  • Theorem 11.14

Lean declarations

\(\mathcal{Z}(0) = 1\).

Theorem 11.19 Gauge Z-Operator Addition
✓
#

Uses

  • Definition 11.17
  • Theorem 11.16
  • Definition 11.17
  • Theorem 11.16

Lean declarations

  • GaugingLDPC.gaugeZOperator_add

For 1-chains \(\alpha \) and \(\beta \), \(\mathcal{Z}(\alpha + \beta ) = \mathcal{Z}(\alpha ) \cdot \mathcal{Z}(\beta )\).

Proof ▶

We have

\[ \mathcal{Z}(\alpha + \beta ) = \prod _{e \in E} Z_e^{(\alpha + \beta )_e} = \prod _{e \in E} Z_e^{\alpha _e + \beta _e}. \]

By Theorem 11.16, \(Z_e^{\alpha _e + \beta _e} = Z_e^{\alpha _e} \cdot Z_e^{\beta _e}\). Thus

\[ \mathcal{Z}(\alpha + \beta ) = \prod _{e \in E} (Z_e^{\alpha _e} \cdot Z_e^{\beta _e}) = \left(\prod _{e \in E} Z_e^{\alpha _e}\right) \cdot \left(\prod _{e \in E} Z_e^{\beta _e}\right) = \mathcal{Z}(\alpha ) \cdot \mathcal{Z}(\beta ) \]

where we used that products distribute over finite sets (in a commutative ring).

Definition 11.22 Gauging Map
✓
#

Uses

  • Definition 5.24
  • Definition 11.17
  • Definition 11.21
  • Definition 11.5
  • Definition 11.12

Lean declarations

  • GaugingLDPC.gaugingMap

The gauging (decoration) map \(\mathfrak {D}: \mathcal{O} \to \mathcal{O} \otimes \mathcal{O}_{\mathrm{gauge}}\) is defined by

\[ \mathfrak {D}(O) := \sum _{\chi : V \to \mathbb {Z}/p\mathbb {Z}} \llbracket O \rrbracket _\chi \otimes \mathcal{Z}(\alpha (\chi )) \]

where:

  • \(\llbracket O \rrbracket _\chi \) is the charge-\(\chi \) component of \(O\) (via multi-site charge projection),

  • \(\alpha (\chi )\) is the unique 1-chain in \(T_1\) solving \(\partial _1(\alpha ) = \chi \) (when \(\chi \) is exact),

  • \(\mathcal{Z}(\alpha )\) is the gauge Z-operator.

The sum is over all charge distributions, implementing the completeness of charge expansion.

Theorem 11.23 Gauging of Zero
✓
#

Uses

  • Definition 11.22
  • Definition 11.22
  • Definition 5.24

Lean declarations

  • GaugingLDPC.gaugingMap_zero

\(\mathfrak {D}(0) = 0\).

Proof ▶

By simplification: for each charge distribution \(\chi \), the charge projection of zero is zero (since the product action maps zero to zero), so \(\llbracket 0 \rrbracket _\chi = 0\). Thus each term in the sum is \(0 \otimes \mathcal{Z}(\alpha (\chi )) = 0\), and the sum of zeros is zero.

Theorem 11.24 Gauging is Additive
✓
#

Uses

  • Definition 11.22
  • Definition 11.22
  • Definition 5.24

Lean declarations

  • GaugingLDPC.gaugingMap_add

For operators \(O_1\) and \(O_2\), \(\mathfrak {D}(O_1 + O_2) = \mathfrak {D}(O_1) + \mathfrak {D}(O_2)\).

Proof ▶

Expanding the definition of \(\mathfrak {D}\), we use that sums distribute over addition. For each \(\chi \), we need to show that \(\llbracket O_1 + O_2 \rrbracket _\chi \otimes \mathcal{Z}(\alpha (\chi )) = \llbracket O_1 \rrbracket _\chi \otimes \mathcal{Z}(\alpha (\chi )) + \llbracket O_2 \rrbracket _\chi \otimes \mathcal{Z}(\alpha (\chi ))\).

First, we establish that the charge projection is additive: \(\llbracket O_1 + O_2 \rrbracket _\chi = \llbracket O_1 \rrbracket _\chi + \llbracket O_2 \rrbracket _\chi \). This holds because for all \(g\), the product action satisfies \(\sigma _g(O_1 + O_2) = \sigma _g(O_1) + \sigma _g(O_2)\) (since the composed automorphism is a ring homomorphism), and scalar multiplication distributes over addition. By the distributivity of tensor product over addition, \((A + B) \otimes C = A \otimes C + B \otimes C\), we obtain the result.

Theorem 11.25 Gauging Uses Completeness
✓
#

Uses

  • Definition 5.24
  • Definition 5.24

Lean declarations

  • GaugingLDPC.gaugingMap_uses_completeness

For any operator \(O\), we have the completeness relation

\[ O = \sum _{\chi : V \to \mathbb {Z}/p\mathbb {Z}} \llbracket O \rrbracket _\chi . \]
Proof ▶

This is precisely the completeness of the charge expansion (Lemma 2), applied to the multi-site setting. The equality follows from the symmetry of the statement by taking the symmetric version of the completeness theorem.

11.6 Remark on Basis Independence

Definition 11.26 Chain Equivalence Modulo Kernel
✓
#

Uses

  • Definition 11.1

Lean declarations

  • GaugingLDPC.ChainEquivMod

Two 1-chains \(\alpha \) and \(\alpha '\) are equivalent modulo \(\ker (\partial _1)\), written \(\alpha \sim \alpha '\), if \(\partial _1(\alpha - \alpha ') = 0\), i.e., their difference lies in the kernel of the boundary map.

Theorem 11.27 Chain Equivalence is Reflexive
✓
#

Uses

  • Definition 11.26
  • Definition 11.26
  • Definition 11.1

Lean declarations

  • GaugingLDPC.chainEquivMod_refl

For any 1-chain \(\alpha \), \(\alpha \sim \alpha \).

Proof ▶

By simplification: \(\partial _1(\alpha - \alpha ) = \partial _1(0) = 0\).

Theorem 11.28 Chain Equivalence is Symmetric
✓
#

Uses

  • Definition 11.26
  • Definition 11.26
  • Definition 11.1

Lean declarations

  • GaugingLDPC.chainEquivMod_symm

If \(\alpha \sim \alpha '\), then \(\alpha ' \sim \alpha \).

Proof ▶

Assume \(\partial _1(\alpha - \alpha ') = 0\). Then \(\partial _1(\alpha ' - \alpha ) = \partial _1(-(\alpha - \alpha ')) = -\partial _1(\alpha - \alpha ') = -0 = 0\).

Theorem 11.29 Chain Equivalence is Transitive
✓
#

Uses

  • Definition 11.26
  • Definition 11.26
  • Definition 11.1

Lean declarations

  • GaugingLDPC.chainEquivMod_trans

If \(\alpha _1 \sim \alpha _2\) and \(\alpha _2 \sim \alpha _3\), then \(\alpha _1 \sim \alpha _3\).

Proof ▶

Assume \(\partial _1(\alpha _1 - \alpha _2) = 0\) and \(\partial _1(\alpha _2 - \alpha _3) = 0\). Note that \(\alpha _1 - \alpha _3 = (\alpha _1 - \alpha _2) + (\alpha _2 - \alpha _3)\). Thus \(\partial _1(\alpha _1 - \alpha _3) = \partial _1(\alpha _1 - \alpha _2) + \partial _1(\alpha _2 - \alpha _3) = 0 + 0 = 0\).

Theorem 11.30 Alpha Map Basis Change
✓
#

Uses

  • Definition 11.21
  • Definition 11.26
  • Definition 11.5
  • Definition 11.21
  • Definition 11.26
  • Theorem 11.9

Lean declarations

  • GaugingLDPC.alphaMap_basis_change

For two spanning bases \(B\) and \(B'\) and any exact charge distribution \(\chi \), the corresponding alpha values are equivalent modulo \(\ker (\partial _1)\):

\[ \alpha _B(\chi ) \sim \alpha _{B'}(\chi ). \]
Proof ▶

Since \(\chi \) is exact, both \(\alpha _B(\chi )\) and \(\alpha _{B'}(\chi )\) are well-defined via the alpha-of-chi construction. Let \(h_B\) be the specification that \(\partial _1(\alpha _B(\chi )) = \chi \) and \(h_{B'}\) be the specification that \(\partial _1(\alpha _{B'}(\chi )) = \chi \). Then:

\[ \partial _1(\alpha _B(\chi ) - \alpha _{B'}(\chi )) = \partial _1(\alpha _B(\chi )) - \partial _1(\alpha _{B'}(\chi )) = \chi - \chi = 0. \]

Thus \(\alpha _B(\chi ) \sim \alpha _{B'}(\chi )\).
