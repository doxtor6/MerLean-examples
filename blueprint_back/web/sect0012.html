12 Def 5: Circuit Unitaries for Gauging

The gauging procedure is implemented by the following circuit unitaries. We work with qudits of dimension \(w\) (a prime), where the computational basis is \(\{ |0\rangle , |1\rangle , \ldots , |w-1\rangle \} \). Arithmetic on basis labels is performed in \(\mathbb {F}_w = \mathbb {Z}/w\mathbb {Z}\).

The Hilbert space of a single qudit is \(\mathbb {C}^w\). We represent states as functions \(\mathbb {Z}_w \to \mathbb {C}\) (the amplitudes in the computational basis).

Definition 12.3 Plus State
✓
#

Uses

  • Definition 12.1

The \(|+\rangle \) state is the equal superposition of all computational basis states:

\[ |+\rangle = \frac{1}{\sqrt{w}} \sum _{j=0}^{w-1} |j\rangle \]

This is the \(+1\) eigenstate of the \(\mathcal{X}\) operator.

✓

Theorem 12.13 Controlled-X Gate on Basis States
✓
#

Uses

  • Definition 12.12
  • Definition 12.11
  • Definition 12.2
  • Definition 12.12
  • Definition 12.11
  • Definition 12.2

Lean declarations

  • GaugingLDPC.controlledXGate_basis

For basis states, \(CX^k (|i_0\rangle \otimes |j_0\rangle ) = |i_0\rangle \otimes |j_0 + i_0 \cdot k\rangle \).

Proof ▶

By extensionality, it suffices to show equality for arbitrary \(i, j\). Using the definitions of controlled-\(X\), tensor state, and basis state, we consider cases on whether \(i = i_0\) and \(j = j_0 + i_0 \cdot k\).

Case 1: If \(i = i_0\) and \(j = j_0 + i_0 \cdot k\), then both sides evaluate to \(1\) by simplification using \(j_0 + i_0 \cdot k - i_0 \cdot k = j_0\).

Case 2: If \(i = i_0\) but \(j \neq j_0 + i_0 \cdot k\), we show \(j - i \cdot k \neq j_0\) by the calculation \(j = (j - i \cdot k) + i \cdot k\). Then both sides evaluate to \(0\).

Case 3: If \(i \neq i_0\), then the control qudit gives \(0\), so both sides are \(0\).

Theorem 12.14 Controlled-X Gate Inverse
✓
#

Uses

  • Definition 12.12
  • Definition 12.12

Lean declarations

  • GaugingLDPC.controlledXGate_inv

The gate \(CX^k\) is invertible with inverse \(CX^{-k}\):

\[ CX^{-k} \circ CX^k = \mathrm{id} \]
Proof ▶

By extensionality, for any \(\Psi , i, j\), we compute \((CX^{-k} \circ CX^k)(\Psi )(i, j) = CX^{-k}(CX^k(\Psi ))(i, j) = CX^k(\Psi )(i, j - i \cdot (-k)) = \Psi (i, (j + i \cdot k) - i \cdot k) = \Psi (i, j)\). The final equality follows by ring arithmetic.

12.6 Edge-Vertex Entangling Unitary

Definition 12.21 Edge-Vertex Entangling Unitary
✓
#

Uses

  • Definition 12.20
  • Definition 12.15

Lean declarations

  • GaugingLDPC.edgeVertexEntanglingUnitary

The edge-vertex entangling unitary \(U_{C\mathcal{X}}\) is:

\[ U_{C\mathcal{X}} := \prod _{v \in \mathcal{G}_0} \prod _{e \in \mathcal{G}_1} (C\mathcal{X}_{v \to e})^{(\partial _1^{\mathcal{G}})_{v,e}} \]

It acts on states as:

\[ U_{C\mathcal{X}} |f_V\rangle |f_E\rangle |f_M\rangle = |f_V\rangle |f_E + \partial _1^T f_V\rangle |f_M\rangle \]

Explicitly, for each edge \(e\), the new edge value is \(f_E(e) + \sum _v f_V(v) \cdot (\partial _1)_{v,e}\). The matter qudits are unchanged.

12.7 Matter-Symmetry Coupling Unitary

Definition 12.22 Matter-Symmetry Coupling Unitary
✓
#

Uses

  • Definition 12.20
  • Definition 12.19

Lean declarations

  • GaugingLDPC.matterSymmetryCouplingUnitary

The matter-symmetry coupling unitary \(U_{C\mathsf{T}}\) is:

\[ U_{C\mathsf{T}} := \prod _{v \in \mathcal{G}_0} C_v \mathsf{T}_{\Sigma (v)} \]

It acts on states as:

\[ U_{C\mathsf{T}} |f_V\rangle |f_E\rangle |f_M\rangle = |f_V\rangle |f_E\rangle |f_M + \sum _v f_V(v) \cdot \mathbf{1}_{\Sigma (v)}\rangle \]

where \(\mathbf{1}_{\Sigma (v)}(m) = 1\) if \(m \in \Sigma (v)\) and \(0\) otherwise. The edge qudits are unchanged.

Theorem 12.24 Plus Projection Normalization
✓
#

Uses

  • Definition 12.23
  • Definition 12.3
  • Definition 12.23
  • Definition 12.3

Lean declarations

  • GaugingLDPC.plusProjection_plusState

\(\langle +|+\rangle = 1\) (normalization).

Proof ▶

Using the definitions of plus projection and plus state:

\[ \langle +|+\rangle = \frac{1}{\sqrt{w}} \sum _{j} \frac{1}{\sqrt{w}} = \frac{1}{\sqrt{w}} \cdot w \cdot \frac{1}{\sqrt{w}} = \frac{w}{w} = 1 \]

The formal proof establishes that \(w {\gt} 0\) (since \(w\) is prime), hence \(\sqrt{w} \neq 0\). We rewrite the constant sum as \(w \cdot (1/\sqrt{w})\) using thecardinality of \(\mathbb {Z}_w\). The equality \((\sqrt{w})^2 = w\) allows us to simplify using field operations.

12.9 Commutativity of Unitaries

Theorem 12.26 Unitaries Commute
✓
#

Uses

  • Definition 12.21
  • Definition 12.22
  • Definition 12.21
  • Definition 12.22

Lean declarations

  • GaugingLDPC.unitaries_commute

The edge-vertex entangling unitary and matter-symmetry coupling unitary commute:

\[ U_{C\mathcal{X}} \circ U_{C\mathsf{T}} = U_{C\mathsf{T}} \circ U_{C\mathcal{X}} \]
Proof ▶

By extensionality, it suffices to show equality for arbitrary \(\Psi , f_V, f_E, f_M\).

The left-hand side first applies \(U_{C\mathsf{T}}\) which transforms \(f_M\), then \(U_{C\mathcal{X}}\) which transforms \(f_E\).

The right-hand side first applies \(U_{C\mathcal{X}}\) which transforms \(f_E\), then \(U_{C\mathsf{T}}\) which transforms \(f_M\).

Both compositions give \(\Psi \) applied to:

  • \(f_V\) (unchanged by both)

  • \(f_E - \sum _v f_V(v) \cdot (\partial _1)_{v,e}\) (edges transformed)

  • \(f_M - \sum _v (\text{if } m \in \Sigma (v) \text{ then } f_V(v) \text{ else } 0)\) (matter transformed)

Since \(U_{C\mathcal{X}}\) modifies only edges using vertices as controls, and \(U_{C\mathsf{T}}\) modifies only matter qudits using vertices as controls, they act on disjoint targets while using the same unchanged control. By simplification, both sides are equal.

Theorem 12.34 Matter-Symmetry Unitary Bijective
✓
#

Uses

  • Definition 12.22
  • Definition 12.31
  • Theorem 12.33
  • Theorem 12.32
  • Theorem 12.33
  • Theorem 12.32
  • Definition 12.31

Lean declarations

  • GaugingLDPC.matterSymmetryCouplingUnitary_bijective

\(U_{C\mathsf{T}}\) is a bijection (hence unitary as an operator).

Proof ▶

We verify both injectivity and surjectivity.

Injectivity: Let \(\Psi _1, \Psi _2\) satisfy \(U_{C\mathsf{T}}(\Psi _1) = U_{C\mathsf{T}}(\Psi _2)\). We show \(\Psi _1 = \Psi _2\) by applying \((U_{C\mathsf{T}})^{-1}\) to both sides. Using the left inverse property \((U_{C\mathsf{T}})^{-1} \circ U_{C\mathsf{T}} = \mathrm{id}\), we calculate:

\[ \Psi _1 = ((U_{C\mathsf{T}})^{-1} \circ U_{C\mathsf{T}})(\Psi _1) = ((U_{C\mathsf{T}})^{-1} \circ U_{C\mathsf{T}})(\Psi _2) = \Psi _2 \]

\[ U_{C\mathsf{T}}((U_{C\mathsf{T}})^{-1}(\Psi )) = (U_{C\mathsf{T}} \circ (U_{C\mathsf{T}})^{-1})(\Psi ) = \mathrm{id}(\Psi ) = \Psi \]
