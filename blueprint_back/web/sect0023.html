23 Thm 3: Distance Preservation Under Gauging

This chapter establishes that the distance of a stabilizer code is preserved (and potentially increased) under gauging when the gauging graph has Cheeger constant \(h(\mathcal{G}) \geq 1\).

Definition 23.1 Original Logical Operator
✓
#

Lean declarations

  • GaugingLDPC.OriginalLogicalOperator

A logical operator of the original code is a structure consisting of:

  • A matter support \(\mathrm{matterSupport} \subseteq V\), the set of vertices where the operator acts non-trivially on matter qudits.

  • A proof that this support is nonempty (logical operators are non-trivial).

Definition 23.2 Original Logical Operator Weight
✓
#

Uses

  • Definition 23.1

Lean declarations

  • GaugingLDPC.OriginalLogicalOperator.weight

The weight of an original logical operator \(L\) is defined as:

\[ \mathrm{weight}(L) := |\mathrm{matterSupport}(L)| \]

✓
#

Uses

  • Definition 23.2
  • Definition 23.1

For any logical operator \(L\), we have \(\mathrm{weight}(L) {\gt} 0\).

Definition 23.4 Gauged Code Weights
✓
#

Lean declarations

  • GaugingLDPC.GaugedCodeWeights

Given a set of logical weights \(\mathcal{L}\) and a function \(\mathrm{equivClassMinWeights}\) that assigns to each logical weight the set of weights achievable in its equivalence class, the gauged code weights are:

\[ \mathrm{GaugedCodeWeights}(\mathcal{L}, \mathrm{equiv}) := \{ n \mid \exists w \in \mathcal{L}, \exists ew \in \mathrm{equiv}(w), n = ew \} \]

This represents all weights of gauged operators coming from logical operators.

Definition 23.5 Gauged Code Distance
✓
#

Uses

  • Definition 23.4

Lean declarations

  • GaugingLDPC.GaugedCodeDistance

The gauged code distance is defined as:

\[ d_g := \inf (\mathrm{GaugedCodeWeights}(\mathcal{L}, \mathrm{equiv})) = \min _L \min _{O \sim _g \mathfrak {D}(L)} |\mathrm{Supp}(O)| \]
Theorem 23.6 Decoration Weight Greater Than or Equal to Original
✓
#

Uses

  • Definition 14.8

Lean declarations

  • GaugingLDPC.decoration_weight_ge_original

For any original logical operator with weight \(w\) and decoration adding edge support of size \(e\), the decorated operator has weight:

\[ w + e \geq w \]

The decorated logical \(\mathfrak {D}(L) = L \otimes \mathcal{Z}(\alpha )\) where \(\alpha \) solves \(\partial _1 \alpha = \chi \) has support that includes both the original matter support and the edge support from \(\alpha \).

Proof ▶

By arithmetic: \(w + e \geq w\) holds for any natural numbers \(w, e \geq 0\).

Theorem 23.7 Flux Equivalence Preserves Matter
✓
#

Lean declarations

  • GaugingLDPC.flux_equiv_preserves_matter

Flux stabilizers \(\mathfrak {B}_p = \prod _{e \in \partial p} \mathcal{Z}_e\) are pure edge operators acting only on gauge qudits. Under flux equivalence \(\sim _{\mathfrak {B}}\), the matter support is preserved. That is, for matter weight \(m\) and any edge weights \(e_1, e_2\):

\[ m + e_2 \geq m \]
Proof ▶

By arithmetic: \(m + e_2 \geq m\) holds for any natural numbers \(m, e_2 \geq 0\).

Theorem 23.8 Vertex Equivalence Weight Non-Decreasing
✓
#

Uses

  • Definition 16.20

Lean declarations

  • GaugingLDPC.vertex_equiv_weight_nondecreasing

Let \(G\) be an expander graph (satisfying \(h(\mathcal{G}) \geq 1\)). For any small subset \(U \subseteq V\):

\[ |\partial U| \geq |U| \]

where \(\partial U\) denotes the edge boundary of \(U\). This means that when applying vertex stabilizers \(\mathfrak {A}_v = \mathsf{T}_{\Sigma (v)} \prod _{e \ni v} \mathcal{X}_e\), the edges gained are at least as many as the matter sites potentially lost.

Proof ▶

This follows directly from the expander property: for any small subset \(U\), the edge boundary satisfies \(|\partial U| \geq |U|\) by the Cheeger constant bound.

Theorem 23.10 Stabilizer Equivalence Preserves Bound
✓
#

Uses

  • Definition 14.8

Lean declarations

  • GaugingLDPC.stabilizer_equiv_preserves_bound

Let \(d\) be the code distance and let \(\mathcal{L}\) be the set of logical weights satisfying \(w \geq d\) for all \(w \in \mathcal{L}\). If \(L\) and \(LS\) are both logical operators (where \(S\) is a stabilizer), then:

\[ \mathrm{weight}(LS) \geq d \]

By Lemma 5, \(\mathfrak {D}(L)\mathfrak {D}(S) = \mathfrak {D}(LS)\) for stabilizers \(S\), so the distance bound is preserved.

Proof ▶

Since \(LS\) is a logical operator with weight \(\mathrm{weight}(LS) \in \mathcal{L}\), and all weights in \(\mathcal{L}\) satisfy \(w \geq d\), we have \(\mathrm{weight}(LS) \geq d\).

Theorem 23.11 Equivalence Class Weight Greater Than or Equal to \(d\)
✓
#

Uses

  • Definition 14.8
  • Definition 16.20

Lean declarations

  • GaugingLDPC.equiv_class_weight_ge_d

Given:

  • A logical operator weight \(w \geq d\)

  • Decoration adding \(e\) edges

  • Vertex equivalence steps with net change \(\Delta \geq 0\)

The final weight satisfies:

\[ w + e + \Delta ^+ \geq d \]

where \(\Delta ^+ = \max (\Delta , 0)\).

Proof ▶

We proceed by establishing two inequalities:

  1. First, \(w + e \geq w\) since decoration adds edges.

  2. Second, \(w + e + \Delta ^+ \geq w + e\) since \(\Delta ^+ \geq 0\).

Combining these with \(w \geq d\), we obtain \(w + e + \Delta ^+ \geq d\) by linear arithmetic.

Theorem 23.12 Distance Preservation Under Gauging
✓
#

Uses

  • Definition 23.5
  • Definition 16.20
  • Definition 14.8
  • Definition 23.4

Lean declarations

  • GaugingLDPC.DistancePreservationUnderGauging

Main Theorem. For a gauging graph \(\mathcal{G}\) with Cheeger constant \(h(\mathcal{G}) \geq 1\) (an expander graph), the gauged code distance satisfies:

\[ d_g \geq d \]

where \(d\) is the distance of the original code and \(d_g := \min _L \min _{O \sim _g \mathfrak {D}(L)} |\mathrm{Supp}(O)|\).

Proof ▶

We unfold the definition of \(d_g\) as \(\inf (\mathrm{GaugedCodeWeights})\).

First, we show the set of gauged code weights is nonempty. Let \(w \in \mathcal{L}\) be a logical weight (which exists by the nonemptiness hypothesis). Then there exists some \(ew \in \mathrm{equivClassMinWeights}(w)\) by the nonemptiness of equivalence classes. Thus \((ew, w, ew)\) is in the gauged code weights set.

We apply the principle that \(d \leq \inf (S)\) if and only if \(d\) is a lower bound for \(S\). For any \(n \in \mathrm{GaugedCodeWeights}\), we decompose \(n\) into its components: there exists a logical weight \(w \in \mathcal{L}\) and an equivalence weight \(ew \in \mathrm{equivClassMinWeights}(w)\) such that \(n = ew\).

By the equivalence bound hypothesis, \(ew \geq w\). By the distance hypothesis, \(w \geq d\). Therefore \(n = ew \geq d\) by transitivity.

Since \(d\) is a lower bound for all elements of \(\mathrm{GaugedCodeWeights}\), we conclude \(d_g = \inf (\mathrm{GaugedCodeWeights}) \geq d\).

Theorem 23.13 Distance Preservation with Expander
✓
#

Uses

  • Definition 23.1
  • Definition 16.20
  • Definition 23.1

Lean declarations

  • GaugingLDPC.DistancePreservationWithExpander

Let \(G\) be an expander graph. For a code with distance \(d\) and set of logical operators \(\mathcal{L}\) (each with weight \(\geq d\)), if:

  • Each logical has a nonempty gauged equivalence class

  • Each weight in an equivalence class is \(\geq \) the logical weight

Then:

\[ \inf \{ w \mid \exists L \in \mathcal{L}, w \in \mathrm{equivClassWeights}(L) \} \geq d \]
Proof ▶

First, we establish that the set of all equivalence class weights is nonempty. Let \(L \in \mathcal{L}\) be a logical operator (which exists by hypothesis). Then there exists \(w \in \mathrm{equivClassWeights}(L)\) by the nonemptiness hypothesis.

We apply the infimum lower bound principle. For any \(n\) in the set of all equivalence class weights, there exists \(L \in \mathcal{L}\) such that \(n \in \mathrm{equivClassWeights}(L)\).

By the weight bound hypothesis, \(n \geq \mathrm{weight}(L)\). By the distance hypothesis, \(\mathrm{weight}(L) \geq d\). Therefore \(n \geq d\) by transitivity.

Since \(d\) is a lower bound for all elements, the infimum is at least \(d\).

Theorem 23.18 Gauged Operator Weight Greater Than or Equal to Logical

Theorem 23.19 Expander Vertex Step Non-Negative
✓
#

Uses

  • Definition 16.20
  • Definition 16.20

Lean declarations

  • GaugingLDPC.expander_vertex_step_nonneg

For an expander graph \(G\) with Cheeger constant \(h(G) \geq 1\), and a sequence of vertex subsets \(U_1, \ldots , U_k\) (each being a small subset), the cumulative net weight change from applying vertex stabilizers is non-negative:

\[ \sum _{i=1}^k \left( |\partial U_i| - |U_i| \right) \geq 0 \]
Proof ▶

We prove this by induction on the list of steps.

Base case: For the empty list, the fold over an empty list starting from accumulator \(0\) returns \(0 \geq 0\).

Inductive step: Suppose the property holds for a list of steps with any non-negative accumulator. For a list \(U :: \mathrm{rest}\):

  1. Since \(U\) is a small subset and \(G\) is an expander, we have \(|\partial U| \geq |U|\) by the Cheeger bound.

  2. Therefore the step contribution \(|\partial U| - |U| \geq 0\).

  3. The new accumulator \(\mathrm{acc} + |\partial U| - |U| \geq 0\) since \(\mathrm{acc} \geq 0\).

  4. By the induction hypothesis, the fold over \(\mathrm{rest}\) starting from this non-negative accumulator is also non-negative.
